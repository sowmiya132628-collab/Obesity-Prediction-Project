# Obesity Prediction using Machine Learning

## Project Overview

This project predicts obesity levels using machine learning techniques based on health and lifestyle attributes.

## Dataset

The dataset contains information such as:

- Gender
- Age
- Height
- Weight
- Family History
- Food Consumption
- Physical Activity
- Water Intake
- Smoking
- Transportation Mode

## Project Workflow

- Data Cleaning
- Label Encoding
- Exploratory Data Analysis
- Outlier Detection
- Feature Selection
- Train-Test Split
- Machine Learning Model Building
- Model Evaluation

## Models Used

- Decision Tree
- Random Forest

## Results

| Model | Accuracy |
|--------|----------|
| Decision Tree | 94.80% |
| Random Forest | 95.51% |

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
